# 羽毛球 Ladder League 管理系统

一个简单易用的羽毛球双打比赛管理系统，支持自动分组、配对、计分和升降级。

## 功能特点

- ✅ 选手管理（添加/删除）
- ✅ 自动分组（4人一组）
- ✅ 自动配对（每人与其他3人各搭档一次）
- ✅ 比赛报分
- ✅ 实时排名（按胜场和净胜分）
- ✅ 自动升降级
- ✅ 移动端友好
- ✅ 数据本地存储

## 快速开始

### 1. 安装依赖

```bash
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

浏览器会自动打开 http://localhost:3000

### 3. 使用流程

1. **管理页面** - 添加选手
2. **管理页面** - 点击"自动分组"创建分组和比赛
3. **报分页面** - 记录每场比赛结果
4. **首页** - 查看实时排名
5. **管理页面** - 所有比赛完成后，点击"升降级并开始新一轮"

## 比赛规则

- 每组4人，两两配对进行组内对抗
- 每人与其他3人各搭档一次，共6场比赛
- 21分制，一局定胜负
- 按胜场数排名，胜场相同则按净胜分排名
- 每组第1名升级，第4名降级

## 部署到线上（免费）

### 使用 Vercel（推荐）

1. 注册 Vercel 账号：https://vercel.com
2. 安装 Vercel CLI：`npm install -g vercel`
3. 在项目目录运行：`vercel`
4. 按提示完成部署

### 使用 Netlify

1. 注册 Netlify 账号：https://netlify.com
2. 拖拽项目文件夹到 Netlify 网站
3. 构建命令：`npm run build`
4. 发布目录：`dist`

## 技术栈

- React 18
- Vite
- LocalStorage（数据存储）

## 注意事项

- 数据存储在浏览器本地，清除浏览器数据会丢失记录
- 建议定期导出数据备份（功能开发中）
- 支持所有现代浏览器和移动设备
